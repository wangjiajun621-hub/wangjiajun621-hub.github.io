초급 (Beginner) — 로지스틱 회귀 낙상 위험 예측

Logistic Regression · 4개 특성 입력 → 혼동행렬 + 계수 시각화

환자 입력 데이터

4개 핵심 임상 변수

모델 설명

Logistic Regression 작동 원리

로지스틱 회귀는 입력 변수들의 선형 결합을 시그모이드 함수로 변환해 0~1 사이의 낙상 확률을 출력합니다.

P(낙상) = σ(β₀ + β₁·age + β₂·balance + β₃·gait + β₄·falls)

📊 샘플 수: 200개 (랜덤 시드 42)

🔀 분할: 학습 80% / 테스트 20%

🎯 레이블: (70세 초과 AND 균형<50) OR 낙상>1

✅ 특징: 해석 가능성 높음, 계수로 각 변수 영향력 확인

    중급 (Intermediate) — 다중 분류기 비교

    Logistic Regression · Decision Tree · Random Forest · SVM — 5-fold CV + ROC 곡선 + 특성 중요도

    추가 특성 입력 (6개)

    중급은 6개 변수를 사용합니다

    분류기 설명

    4가지 알고리즘 비교
    Logistic Regression
    선형 결정 경계. 해석 가능성 최고. 비선형 패턴에 취약. max_iter=1000
    Decision Tree
    트리 구조 규칙 학습. 직관적이지만 과적합 위험. random_state=42
    Random Forest ⭐ 권장
    100개 트리 앙상블. 과적합 방지, 특성 중요도 제공. 높은 AUC.
    SVM (Support Vector Machine)
    마진 최대화 결정 경계. 고차원 데이터에 강함. probability=True
    📐 데이터: 300샘플, 6특성, 5-fold CV
    🎯 레이블: 초급과 동일 + 노이즈 15%
    CV Score vs 테스트 정확도 비교
    ROC 곡선 (AUC)
    Random Forest 특성 중요도 (Feature Importance)

    모델 성능 상세

    고급 (Advanced) — 회귀 기반 재활 결과 예측

    Linear Regression · Random Forest · Gradient Boosting · GridSearchCV · 학습 곡선

    초기 임상 상태 (7개 특성)

    rehabilitation outcome prediction

    고급 모델 설명

    회귀 모델 + 하이퍼파라미터 튜닝

    목표 변수(y) = 재활 기능 점수 (0~100)

    y = 100 - pain×5 + duration×2
        - age×0.3 + compliance×30 + ε

    🔧 GridSearchCV 튜닝:

    max_depth: [5, 10, 15]
    n_estimators: [50, 100, 200]
    cv=5, n_jobs=-1

    📊 평가 지표: RMSE · MAE · R²

    📈 학습 곡선: Train vs Test R² by sample size
